Logiciel de Gestion des Absences des Élèves - ESIGELEC

Septembre 2022 - Mai 2023

Description du Projet

Développement d'une application web complète pour la gestion des absences des étudiants de l'ESIGELEC. Le projet visait à moderniser et automatiser le processus de suivi des présences.

Tâches Réalisées

  • Rédaction du cahier des charges et analyse des besoins avec les parties prenantes
  • Conception de la base de données relationnelle (MySQL) pour stocker les informations des étudiants, cours et absences
  • Développement du backend avec Java et Spring Boot
  • Création d'une interface utilisateur responsive avec HTML/CSS et Thymeleaf
  • Implémentation d'un système d'authentification pour différents rôles (admin, professeurs, étudiants)
  • Développement de fonctionnalités de génération de rapports d'absence automatiques
  • Mise en place d'un système de notification par email pour les absences non justifiées
  • Tests unitaires et d'intégration avec JUnit

Technologies Utilisées

Java Spring Boot MySQL HTML/CSS Thymeleaf JUnit Git Maven

Fonctionnalités Clés

  • Interface d'administration pour la gestion des utilisateurs et des cours
  • Système de pointage numérique pour les présences
  • Tableau de bord personnalisé pour chaque type d'utilisateur
  • Génération automatique de rapports d'absence
  • Système de justification d'absence en ligne
  • Historique complet des absences par étudiant

Résultats

L'application a permis de réduire de 70% le temps de traitement des absences et d'améliorer significativement le suivi des étudiants. Le système est actuellement utilisé quotidiennement par plus de 2000 étudiants et 100 professeurs.

Compétences Développées

  • Gestion de projet et méthodologie Agile
  • Développement full-stack Java
  • Conception de bases de données
  • Sécurité des applications web
  • Tests et déploiement d'applications
