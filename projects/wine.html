Prédiction Qualité du Vin

Mars 2024 - Avril 2024

Description du Projet

Développement d'un modèle de Machine Learning pour prédire la qualité des vins basé sur leurs caractéristiques physico-chimiques.

Tâches Réalisées

  • Collecte et préparation d'un dataset de plus de 1500 échantillons de vin
  • Analyse exploratoire des données avec Python et Jupyter Notebook
  • Feature engineering et sélection des variables pertinentes
  • Développement et comparaison de plusieurs modèles ML (Random Forest, XGBoost, SVM)
  • Optimisation des hyperparamètres avec GridSearchCV
  • Validation croisée et évaluation des performances du modèle
  • Déploiement du modèle via une API REST
  • Création d'une interface utilisateur avec Streamlit

Technologies Utilisées

Python Scikit-learn XGBoost Pandas Numpy Matplotlib Streamlit Flask

Résultats

Le modèle final atteint une précision de 89% dans la prédiction de la qualité du vin, avec une interface utilisateur intuitive permettant aux œnologues de prédire rapidement la qualité d'un vin basé sur ses caractéristiques.

Impact

Ce projet a permis d'automatiser le processus d'évaluation de la qualité du vin, réduisant le temps nécessaire pour l'évaluation de 75% tout en maintenant une haute précision.
